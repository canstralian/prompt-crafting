## Settings Pages Structure

### Account Settings
- Profile (name, email, avatar)
- Password & security
- Connected accounts
- Delete account

### Team Settings (Pro/Team only)
- Team members
- Roles & permissions
- Invite link
- Team name & avatar

### Billing Settings
- Current plan
- Payment method
- Invoices
- Usage stats
- Cancel/downgrade

### Preferences
- Brand defaults
  - Default tone
  - Default output format
  - Custom constraints
- Model preferences
  - Preferred model
  - Default temperature
  - Max length
- Export preferences
  - Default format
  - Include metadata

## Route Guards

### Public Routes
No authentication required:
- `/`, `/templates`, `/library`, `/learn/*`, `/pricing`, `/auth`

### Protected Routes
Require authentication:
- `/app/*`

### Admin Routes
Require admin role:
- `/app/admin`

### Team Routes
Require team plan:
- `/app/workspace`

```tsx
// Route protection example
<Route path="/app" element={<ProtectedRoute />}>
  <Route path="dashboard" element={<DashboardPage />} />
  <Route path="library" element={<LibraryPage />} />
  <Route path="admin" element={
    <AdminRoute>
      <AdminPage />
    </AdminRoute>
  } />
  <Route path="workspace" element={
    <TeamRoute>
      <WorkspacePage />
    </TeamRoute>
  } />
</Route>
```

## Deep Linking

### Prompt Links
```
/app/prompts/:id                    # Opens prompt in builder
/app/prompts/:id?version=3          # Opens specific version
/app/prompts/:id?section=constraints # Scrolls to section
```

### Search Links
```
/app/library?q=marketing            # Pre-filtered search
/templates?category=code&tone=technical # Filtered templates
```

### Onboarding Links
```
/app/prompts/new?outcome=writing    # Pre-selected onboarding step
/app/prompts/new?template=abc123    # Start from template
```
